#### 文件结束符
Windows系统下键入"Ctrl + z"作为结束符; Linux及Mac OS以"Ctrl + d"作为结束符
#### 静态类型
C++是静态类型(statically typed)语言, 在编译时执行类型检查
#### 类型含义
类型确定了数据和操作在程序中的意义. 如`i = i + j;`若i和j是整型
则这条语句表示算术"+"运算; 若i和j为string类型，则该语句表示字符串拼接运算
#### 内置类型
布尔型、字符型、整型、浮点型
#### 算术类型
用来表示整数、浮点数、单个字符和布尔值的类型
#### 整型
表示整数、字符和布尔值的算术类型合称为整型.
short、int和long类型都表示整型值, 存储空间的大小不同. 一般，short为半个机器
字长，int为一个机器字长，long为一个或两个字长(在32位机器中int和long类型字长通常相同)
#### 类型符号
除bool类型外, 整型可以是__带符号的(signed)__也可以是__无符号的(unsigned)__.
整型int、short和long都默认为带符号型, 要获得无符号型则需指定为unsigned, 比如unsigned
 long, unsigned int可以简写为unsigned.
#### 整型的越界赋值
对__unsigned__类型越界赋值时，编译器会将该值对unsigned类型的可能取值求模，然后取所得值. 比如
8位的unsigned char, 其取值范围从0到255. 若用-1赋值则结果为255(-1对256求模);
对__signed__类型越界赋值时, 通常也是求模,但不是绝对，由编译器决定实际赋的值.
#### 浮点型
类型float、double和long double分别表示单精度浮点数、双精度浮点数和扩展精度浮点数. 一般float用
一个字(32位)表示, double用两个字表示, long double类型用三个或四个字表示. 类型的取值范围决定过了
浮点数所含的有效数字位数.
对于实际程序来说, float类型精度通常是不够的---float只能保证6位有效数字, 而double至少保证10位
有效数字，能蛮子大多数计算的需要.
#### 字面值常亮
像0这样的值在程序中称为字面值常量
#### 整型字面值规则
定义字面值整数常量可以使用十进制、八进制(0)、十六进制(0x或0X)
字面值整数常量的类型默认为int或long类型. 其精度类型决定于字面值---其值大于int最大值就为long，
否则为int. 通过增加后缀, 可以强制将其转换为long、unsigned或unsigned long类型
	128u /* unsigned */	1024UL /* unsigned long */
	1L	/* long */		8Lu		/* unsigned long */
注: 没有short型字面值常量
